Обзор
=====

`ar_ex_sys_worker` — это модуль для отправки данных из AR (весовой/проходной
контур) во внешние системы по REST API. Он:
- собирает записи из БД (акты, проходы КПП, инциденты, фото/видео),
- приводит данные к формату внешней системы,
- отправляет и логирует результат,
- при необходимости выполняет обратные операции (удаление/обновление актов,
  синхронизация справочников).

Ключевые сущности:
- `DataWorker` — базовый отправщик данных (забор данных, форматирование,
  отправка, логирование).
- `ActsWorker` и наследники — отправка актов (records) во внешнюю систему.
- `SignAll*` — интеграции с Signall (акты, КПП, лифты, клиенты, авто).
- `ASU*` — интеграции с ASU (акты и маршруты).
- `mixins` — переиспользуемые блоки: URL‑строитель, авторизация,
  SQL‑команды, логирование, работа с фото/видео.

Базовые зависимости:
- `requests` — HTTP клиент.
- `wsqluse` — обёртка для SQL‑доступа и декораторы.
- `pytz`, `tzlocal` — работа с часовыми поясами.
