ar_ex_sys_worker
================

Модуль для взаимодействия AR с REST API внешних систем. Предназначен для
разработчиков: описывает архитектуру, классы‑воркеры и механики обмена
данными (акты, КПП, фото/видео).

Ключевые возможности
-------------------
- Отправка актов из БД AR во внешние системы (Signall/ASU).
- Логирование отправок в `ex_sys_data_send_reports`.
- Работа с фото/видео (base64) и вложение в payload.
- Token‑авторизация и логин/пароль для разных интеграций.
- Дополнительные операции: обновление/удаление актов, импорт справочников.

Структура проекта
-----------------
- `ar_external_sys_worker/main.py` — основные воркеры и логика обмена.
- `ar_external_sys_worker/mixins.py` — миксины (SQL, auth, логирование).
- `ar_external_sys_worker/tests/` — примеры интеграционных тестов.
- `docs/` — полная документация для разработчиков.

Установка
---------
```
pip install -r requirements.txt
```

Тесты
-----
Тесты требуют доступ к БД и внешним системам, поэтому по умолчанию не
запускаются. Сценарии лежат в `ar_external_sys_worker/tests/`.
